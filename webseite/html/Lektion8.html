Lektion 8: Einführung in CSS

CSS (Cascading Style Sheets) ist die Sprache, die das Aussehen von HTML-Elementen bestimmt. Während HTML die Struktur definiert, definiert CSS das Design: Farben, Schriftarten, Abstände, Layout und Animationen. CSS folgt dem Prinzip der Trennung von Inhalt und Darstellung. Es gibt drei Möglichkeiten, CSS einzubinden: inline, internal oder external. CSS-Regeln bestehen aus Selektoren und Deklarationsblöcken. Das Cascading-Prinzip bestimmt, welche Stile bei Konflikten angewendet werden. CSS macht Webseiten erst richtig schön und benutzerfreundlich. 

<!DOCTYPE html>
<html lang="de">
<head>
    <title>CSS Grundlagen</title>
    <style>
        /* Internal CSS */
        body {
            font-family: Arial, sans-serif;
            background-color: #f0f0f0;
            margin: 0;
            padding: 20px;
        }
        
        h1 {
            color: #2c3e50;
            text-align: center;
            border-bottom: 2px solid #3498db;
        }
        
        .highlight {
            background-color: #f39c12;
            color: white;
            padding: 5px 10px;
            border-radius: 5px;
        }
        
        #special-text {
            font-size: 18px;
            font-weight: bold;
            color: #e74c3c;
        }
    </style>
</head>
<body>
    <h1>Willkommen zu CSS!</h1>
    
    <p>Dies ist ein normaler Absatz.</p>
    
    <p class="highlight">Dieser Text hat eine besondere Hervorhebung.</p>
    
    <p id="special-text">Dieser Text hat eine eindeutige ID.</p>
    
    <p style="color: green; font-style: italic;">
        Dieser Text verwendet Inline-CSS.
    </p>
    
    <div>
        <h2>CSS-Eigenschaften</h2>
        <ul>
            <li>Farben (color, background-color)</li>
            <li>Schriftarten (font-family, font-size)</li>
            <li>Abstände (margin, padding)</li>
            <li>Rahmen (border)</li>
        </ul>
    </div>
</body>
</html>

Bildbeschreibung des beispiel:

Selektoren bestimmen, welche Elemente gestylt werden: Elementname (h1), Klasse (.highlight), ID (#special-text). Deklarationen bestehen aus Eigenschaft und Wert, getrennt durch einen Doppelpunkt. Internal CSS steht im <style>-Tag im , External CSS in separaten .css-Dateien. Inline-CSS hat die höchste Priorität, sollte aber sparsam verwendet werden. Die Kaskadierung löst Konflikte zwischen verschiedenen CSS-Regeln.
