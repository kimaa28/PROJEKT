Lektion 5: Links und Navigation

Links sind das Herzstück des World Wide Web und werden mit dem <a>-Tag erstellt. Das href-Attribut gibt das Ziel des Links an - das kann eine andere Webseite, eine E-Mail-Adresse, eine Telefonnummer oder ein Anker innerhalb der Seite sein. Links können absolut (komplette URL) oder relativ (bezogen auf die aktuelle Seite) sein. Das target-Attribut bestimmt, wo der Link geöffnet wird. title-Attribute bieten zusätzliche Informationen beim Hovern über den Link. Gute Linktexte sind beschreibend und aussagekräftig für die Barrierefreiheit. 

<!DOCTYPE html>
<html lang="de">
<head>
    <title>Links und Navigation</title>
</head>
<body>
    <h1>Verschiedene Arten von Links</h1>
    
    <nav>
        <a href="#abschnitt1">Sprung zu Abschnitt 1</a> |
        <a href="#abschnitt2">Sprung zu Abschnitt 2</a>
    </nav>
    
    <p><a href="https://www.example.com" target="_blank" 
       title="Öffnet externe Seite">Externe Webseite</a></p>
    
    <p><a href="mailto:info@example.com">E-Mail senden</a></p>
    
    <p><a href="tel:+49123456789">Anrufen</a></p>
    
    <p><a href="../andere-seite.html">Relative Verlinkung</a></p>
    
    <h2 id="abschnitt1">Abschnitt 1</h2>
    <p>Inhalt des ersten Abschnitts...</p>
    
    <h2 id="abschnitt2">Abschnitt 2</h2>
    <p>Inhalt des zweiten Abschnitts...</p>
</body>
</html>

Bildbeschreibung des beispiel:

Das <nav>-Element kennzeichnet Navigationsbereiche für bessere Semantik. target="_blank" öffnet Links in einem neuen Tab/Fenster. Anker-Links (#) springen zu Elementen mit entsprechender ID auf derselben Seite. mailto: und tel: Links öffnen E-Mail-Programme bzw. Telefon-Apps. Relative Links sind praktisch für interne Seitenstrukturen.
