Lektion 7: Listen erstellen

HTML bietet drei Haupttypen von Listen: geordnete Listen (<ol>), ungeordnete Listen (<ul>) und Beschreibungslisten (<dl>). Geordnete Listen verwenden Nummerierung, ungeordnete Listen verwenden Aufzählungszeichen. Jeder Listeneintrag wird mit <li> erstellt. Listen können verschachtelt werden für hierarchische Strukturen. Beschreibungslisten bestehen aus Begriffen (<dt>) und Beschreibungen (<dd>). Listen sind semantisch wichtig und helfen bei der Strukturierung von Inhalten. CSS kann das Aussehen von Listen vollständig anpassen. 

<!DOCTYPE html>
<html lang="de">
<head>
    <title>Listen in HTML</title>
</head>
<body>
    <h1>Verschiedene Listentypen</h1>
    
    <h2>Ungeordnete Liste (Aufzählung)</h2>
    <ul>
        <li>Äpfel</li>
        <li>Bananen</li>
        <li>Orangen</li>
    </ul>
    
    <h2>Geordnete Liste (Nummerierung)</h2>
    <ol>
        <li>HTML lernen</li>
        <li>CSS verstehen</li>
        <li>JavaScript üben</li>
    </ol>
    
    <h2>Verschachtelte Liste</h2>
    <ul>
        <li>Frontend-Technologien
            <ul>
                <li>HTML</li>
                <li>CSS</li>
                <li>JavaScript</li>
            </ul>
        </li>
        <li>Backend-Technologien
            <ol>
                <li>PHP</li>
                <li>Python</li>
                <li>Node.js</li>
            </ol>
        </li>
    </ul>
    
    <h2>Beschreibungsliste</h2>
    <dl>
        <dt>HTML</dt>
        <dd>Hypertext Markup Language - strukturiert Webinhalte</dd>
        
        <dt>CSS</dt>
        <dd>Cascading Style Sheets - gestaltet das Aussehen</dd>
    </dl>
</body>
</html>

Bildbeschreibung des beispiel:

<ul> für Listen ohne bestimmte Reihenfolge (Shopping-Listen, Features). <ol> für Listen mit wichtiger Reihenfolge (Anleitungen, Rankings). Verschachtelte Listen erstellen Hierarchien durch Listen innerhalb von <li>-Elementen. <dl>, <dt> und <dd> sind perfekt für Glossare und Definitionen. Listen können mit CSS-Eigenschaften wie list-style-type angepasst werden.
