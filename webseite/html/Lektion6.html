Lektion 6: Bilder einbinden

Bilder werden mit dem <img>-Tag eingebunden und sind selbstschließende Elemente. Das src-Attribut gibt den Pfad zum Bild an, alt-Attribut beschreibt das Bild für Screenreader. Immer ein alt-Attribut verwenden - es ist essentiell für die Barrierefreiheit! width und height können die Bildgröße kontrollieren, aber CSS ist dafür besser geeignet. Das title-Attribut zeigt einen Tooltip beim Hovern an. Moderne Webseiten nutzen oft das -Element für responsive Bilder. Bildformate wie WebP bieten bessere Kompression als klassische JPG/PNG. 

<!DOCTYPE html>
<html lang="de">
<head>
    <title>Bilder einbinden</title>
</head>
<body>
    <h1>Verschiedene Arten der Bildeinbindung</h1>
    
    <h2>Einfaches Bild</h2>
    <img src="landschaft.jpg" alt="Schöne Berglandschaft bei Sonnenuntergang" 
         title="Foto aus den Alpen">
    
    <h2>Bild mit Größenangabe</h2>
    <img src="logo.png" alt="Firmenlogo" width="200" height="100">
    
    <h2>Responsive Bild mit Picture-Element</h2>
    <picture>
        <source srcset="bild-gross.webp" media="(min-width: 800px)">
        <source srcset="bild-mittel.webp" media="(min-width: 400px)">
        <img src="bild-klein.jpg" alt="Produktfoto eines Smartphones">
    </picture>
    
    <h2>Bild als Link</h2>
    <a href="grosses-bild.jpg">
        <img src="thumbnail.jpg" alt="Klicken für größere Ansicht">
    </a>
</body>
</html>

Bildbeschreibung des beispiel:

Das alt-Attribut ist nicht optional - es macht Bilder für sehbehinderte Menschen zugänglich. Bei dekorativen Bildern kann alt="" (leer) verwendet werden. Das <picture>-Element lädt je nach Bildschirmgröße unterschiedliche Bilder. srcset ermöglicht die Bereitstellung verschiedener Bildauflösungen. Bilder in Links werden automatisch mit einem blauen Rahmen versehen (kann per CSS entfernt werden).
